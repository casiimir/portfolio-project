FaceBox

Questa semplice web app combina la potenza del Machine Learning a quella dell'elaborazione delle immagini. Grazie alle API di Clarifai è infatti possibile catturare ogni espressione facciale, da quelle ben visibili a quelle meno visibili.

Cosa ho imparato da questo progetto: React (components - props - states - children - Router - DOM - Events), Npm, JSON, API, Asyncronous, Tachyons, e tanto Html/Css e Javascript!

Movie Finder

Un semplicissimo motore di ricerca Movies, scritto in in React e basato su OMDb RESTful API. Interattivo al massimo e ottimizzato per la velocità e la precisione di ricerca

Questo progetto è stato utile al fine di padroneggiare:
Fetch API, Responsive Design, React (high-order functions - Regular Expressions), Tachyons, Npm, Git & GitHub.

Terminal

Un tuffo nel passato, quando ancora i computer non disponevano di una bella interfaccia grafica. Questo progetto, ancora in corso, riproduce un terminale funzionante e nasce compre primo esperimento di Vue.js

Con questo progetto ho potuto apprendere e usare: Vue.Js, Vuetify, DOM interaction, Reactive Data and Events, Routing, Advanced Javascript, HTML5 & CSS3
